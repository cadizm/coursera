ML:Errata: Week 5

From Coursera

Jump to: navigation, search

In the video titled Backpropagation intuition (duration 12:44 minutes) on the time 4:44 the last term for the calculation for z31 (three-color handwritten formula) must include a22, instead of a21.

In the video titled Backpropagation intuition (duration 12:44 minutes) on the slide 3, cost(i) is not well defined. The second term should be (1−y(i))log(1−hθ(x(i))) and not (1−y(i))logθ(x(i)) as shown. The discrepancy continues throughout the video.

In the video titled Implementation Note (duration 7:47 minutes), in the example starting on time 02:04, the neural network is shown having three layers (one input, one hidden and one output), where as three θ terms are defined. For the ensuing discussion to be correct, one more hidden layer is to be added to the network so that s1=10,s2=10,s3=10 and s4=1. The diagram needs to be similarly modified.

In lecture slides (file docs_slides_Lecture9.pdf) page 5 of 33: the final term in the expression for J(θ) has a subscript i missing i.e. θ(l)j becomes θ(l)ij, and i,j index allows every element in array l to contribute to the matrix norm. This matches the final equation on page 3.

Error in video "Backpropagation Algorithm" at 0:14-1.00: indices for Θ should be Θ(l)ij for both in the cost function and in the partial derivative.

Regarding calculation of the theta gradients: Error in Lecture slides, bottom of page 8, and video "Backpropagation Algorithm" (11:59) at 1:30 and at 10:47:

Regarding Gradient Computation for forward propagation on page 7/33: a(1) = x where given one training example(x,y) line omits a necessary step: add a subscript 0 superscript(1) the bias unit as was done in in step3 for a superscript(2) and step5 for a superscript(3).

D(l)i,j :=1m(Δ(l)i,j+λΘ(l)i,j) If j≠0 NOTE: Typo in lecture slide omits outside parentheses. This version is correct and consistent with the ex4.pdf file (top of Page 12).

9 - 4 Implementation Detail (Lecture9.pdf p. 8): Name collision! The loop/training example index "i" is overloaded with the node index for the next layer.
