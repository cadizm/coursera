ML:Errata: Week 7

From Coursera

Jump to: navigation, search

In 'Optimization Objective', starting at 9:25, the SVM regularization term should start from j=1, not j=0.

In 'Optimization Objective', starting at 13:37, the SVM regularization term should be summing over j instead of i: ∑j=1nθj2.

In 'Large Margin Intuition', starting from 1:04, should be drawn Z <= -1, for graphic on the right.

In 'Mathematics Behind Large Margin Classification', starting from 11:22, second condition should be p(i)⋅∣∣θ∣∣≤−1 if y(i)=0 instead of y(i)=1. This persists also in the quiz.

In 'Mathematics Behind Large Margin Classification', at 16:33, Dr. Ng writes towards the right of the slide that for a vertical decision boundary, p(1).∣∣∣∣θ∣∣∣∣>0, while it should be p(1).∣∣∣∣θ∣∣∣∣>1.

In 'Using An SVM', at 13:51, Dr. Ng writes θ=K instead of class y=K

In programming assignment section Files included in this exercise: svmTrain.m - SVM rraining function should be svmTrain.m - SVM training function

In programming assignment section 1.2.1 Gaussian Kernel in file ex6.m is: "Gaussian Kernel between x1 = [1; 2; 1], x2 = [0; 4; -1], sigma = 0.5 (this value should be about 0.324652)", but should be sigma = 2

In programming assignment section 1.2.3 Example Dataset 3, close to the end of the 3rd paragraph in "...if you try each of the 8 values listed above for C and for σ²..." it should just read "...for C and for σ..." .

In programming assignment section 1.2.3 Example Dataset 3 in file dataset3Params.m, EX6PARAMS should be dataset3Params.

In programming assignment section 1.2.3 Example Dataset 3 and section 2.3 Training SVM for Spam Classification, 'a SVM' should be 'an SVM'.

In ex6.pdf, "rraining" should be "training" on page 1.
